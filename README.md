# 🏦 Analyse des Risques de Crédit Bancaire

## 🎯 Objectif du projet
Ce projet personnel a pour but d'analyser un jeu de données de clients bancaires afin d'identifier les facteurs de risque liés au défaut de paiement. 

## 🛠️ Technologies utilisées
* **Base de données :** Oracle SQL (Modélisation et requêtes d'extraction)
* **Programmation :** Python
* **Bibliothèques :** SQLAlchemy (Connexion BDD), Pandas (Nettoyage et manipulation), Seaborn & Matplotlib (Visualisation)

## 📊 Principaux résultats
L'analyse exploratoire a mis en évidence une forte corrélation entre le montant du crédit accordé et le risque de défaut de paiement.

* Conception de schéma relationnel et importation de données brutes.
* Mise en place d'un pipeline d'extraction de données via SQLAlchemy.
* Nettoyage de données (gestion des valeurs nulles).
